**RO-LE-MA** (Robot Learning Machine) is an open-source system that turns any name, product, or object into a **readable 3-letter code**.
Eiffel Tower -> EIF-TOW
Liberty Statute -> LIB-STA
Coca-Cola Can -> COC-COL-CAN
Robot Learning Machine -> ROB-LEA-MAC
